# CSE_5525_final

1. run backend
* cd gui/server
* activate your virtual env
* pip install flask flask-cors 
* python app.py 

2. run front-end
* conda install -c conda-force nodejs
* cd gui/frontend/
* rm -f node_modules/
* rm package-lock.json
* npm cache clean --force
* npm install 
* npm install --save vuex
* npm install d3@4 --save 
* npm run dev  # to run it
